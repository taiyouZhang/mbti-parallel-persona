> "如果我是另一种性格，这件事我会怎么想？"
>
> 每个深夜 emo 的瞬间，每个纠结到头秃的时刻，你都想过这个问题吧。
>
> 现在，我们给你一个**平行世界的回答**。
> 用你选定的人格，陪你聊、陪你丧、陪你做每一个小决定。

不分析你、不定义你、不说教——就是一个**懂你性格的朋友**，用你的方式陪你说两句话。

---

## ✨ 它能做什么

- 🎭 **第一次使用**：选定你的 MBTI 人格
- ⚡ **之后随时** `/mbti`：用你的人格口吻，秒回当下的事——短、准、懂你
- 🎲 **昵称随场景切换**：同一个 INFP，伤心时叫「流泪猫猫头」，温柔时叫「小蝴蝶」

---

## 🔧 How it works

1. 首次使用设定你的 MBTI
2. 触发时读取你正在聊的内容
3. 1-2 句话，用你的性格直接回——短、有劲、情绪到位

不解释认知功能，不当心理咨询师，不输出大段分析。性格全在语气里。
