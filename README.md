# 🪞 mbti-parallel-persona

> "如果我是另一种性格，这件事我会怎么想？"
>
> 每个深夜 emo 的瞬间，每个纠结到头秃的时刻，你都想过这个问题吧。
>
> 现在，我们给你一个平行世界的回答 🌍
> 用你选定的人格，陪你聊、陪你丧、陪你做每一个小决定。

你的 MBTI 不是标签，是你的赛博闺蜜/损友/树洞 💬

## ✨ 它能做什么

- 🎭 第一次使用：选定你的 MBTI 人格
- ⚡ 之后随时输入 `/mbti`：用你的人格口吻，秒回当下的事——短、准、懂你

## 👀 看看效果

```
You: 我老板今天又在会上否定了我的方案
You: /mbti

INFP(流泪猫猫头):
他否的是方案又不是你，别往心里去。先去散个步，回来再说。
```

```
You: 同事背后说我坏话被我知道了
You: /mbti

ENTJ(指挥官):
影响到你利益了吗？没有就别理，有的话直接当面摊开，你时间比这值钱。
```

```
You: 周末有人约我但我想宅家
You: /mbti

ENFP(修勾):
去啊去啊！万一超好玩呢！不好玩你再跑回来躺也不迟嘛。
```

## 📦 Install

```bash
npx skills add github:taiyouZhang/mbti-parallel-persona
```

## 🎯 Trigger

| Command | Action |
|---------|--------|
| `/mbti` | 用你的人格口吻回应当前话题 |

## 🔧 How it works

1. 首次使用设定你的 MBTI
2. 触发时读取你正在聊的内容
3. 1-2 句话，用你的性格直接回——短、有劲、情绪到位

## 💛 写在最后

- 🌟 **每个你都是独一无二的，没有人可以被四个字母定义**
- 🤝 MBTI 在这里只是你的朋友、你的闺蜜——陪你说说话，给你一个角度
- 💭 它提供的永远只是建议，不是答案
- 👆 做决定的人，始终是你自己
